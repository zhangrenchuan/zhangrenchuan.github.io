CSS简单性能优化

  1. 慎重选择高性能样式
    • 合理安排Selectore
    • 高消耗属性要注意:
      1. box-shadows边框阴影;
      2. border-radius盒子圆角;
      3. transparency透明度;
      4. transforms变形;
      5. CSS filters 滤镜（性能杀手）
  1. 避免过分的重排 (Reflow)
    • 浏览器会重新计算布局位置与大小
    • 常见重排元素: width, height, left, top, right, bottom, padding, margin, display, border, position, font-size, float, text-align, overflow, clear, white-space, min-height
  2. 避免过分重绘 (Repaints)
    • 常见重绘元素: color, border-style, visibility, background, outline, borde-radius, box-shadow
  3. 衡量属性和布局消耗代价
  4. 探索W3C的性能优化新规范
  5. 用测试数据判断优化策略
  6. 多利用GPU性能
